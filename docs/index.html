Features

Built to get out of your way.

Typography

A page, not a text box

Bundled monospace, centered column, muted syntax.

Ctrl+E

Preview on demand

Flip between source and rendered with one key.

Ctrl+Shift+F

Paragraph focus

Fade everything except the line you're in.

KaTeX · Mermaid

Math & diagrams

Rendered in preview. Nothing to configure.

Ctrl+Shift+H / P

Export HTML & PDF

Self-contained HTML. Print-ready A4 PDF.

Tauri · Rust

Native Linux

Small binary. Real file associations. No Electron.

Principles

A quiet page.

No chrome. No toolbars, ribbons, or sidebars.
Keyboard first. Mouse supported, never required.
One theme, locked. A single light palette. The product has a look.
Stillness. No popups, no toasts, no modals.
Install

Two files. One minute.

Both formats ship the same binary and register .md as an associated file type.

Or build from source: pnpm install && pnpm tauri build.

AppImage · any Linux

# download
curl -LO https://github.com/krill-software/text-editor/releases/download/v0.1.1/Markdown_0.1.1_amd64.AppImage

# make executable & run
chmod +x Markdown_0.1.1_amd64.AppImage
./Markdown_0.1.1_amd64.AppImage

Debian / Ubuntu

sudo dpkg -i Markdown_0.1.1_amd64.deb
krill-text-editor path/to/file.md
